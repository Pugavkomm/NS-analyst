# NS analyst

## Install requirements

1. ```python3 -m venv env```
1. ```source env/bin/activate```
1. ``` pip3 install -r requirements.txt ```

## Scripts

В папке `scripts` находится скрипт ```convert_ui_to_py.bash``` для конвертирования файлов из ```*.ui``` в ```*.py```

## Documentation

```./docs/```

``` make html ```
